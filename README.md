# Gestor de Estudos (GestEstudos)

Sistema para organização de rotina acadêmica, permitindo o cadastro de matérias, controle de tempo e acompanhamento de desempenho.

## Funcionalidades Principais
- **Dashboard:** Visualização de cronograma e próximo estudo.
- **Gestão:** Cadastro, edição e exclusão de planos de estudo.
- **Relatório de Desempenho:** Métricas de horas estudadas e média diária.
- **Filtros:** Busca inteligente por matéria ou descrição.

## Tecnologias
- **Frontend:** Next.js 15+, Tailwind CSS, TypeScript.
- **Backend:** Next.js Route Handlers (API).
- **Banco de Dados:** Supabase (PostgreSQL).

## Como Executar
1. Instale as dependências: `npm install`
2. Configure o `.env.local` com as chaves do Supabase.
3. Rode o projeto: `npm run dev`
